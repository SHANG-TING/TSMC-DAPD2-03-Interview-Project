# 1_ENTRY_LEVEL

## 如何啟動執行整個專案

建議切換到 node >= 20 的版本，再執行以下指令

```
npm ci
npm start
```

## 如何驗證功能是否正確

打開 http://localhost:4200



## 簡易的專案架構說明

```
- apps/portal/src/app
- feature/dashboard
  - widgets
    - grid-widget.component.ts
    - text-widget.component.ts
  dashboard.component.ts
- shared/ui/gridbox
```
